BookWise

BookWise é uma plataforma de avaliação de livros que foi desenvolvida durante o bootcamp Ignite da Rocketseat. Com apenas um Figma precisávamos desenvolver essa aplicação completa Full Stack com Next.js.

Next.js, Next Auth, Stitches, Radix, TypeScript, Prisma, React Query
TalentForge

TalentForge é uma aplicação web que ajuda os candidatos a emprego a prepararem-se para entrevistas, gerando perguntas e respostas baseadas na posição a que se candidatam e nas competências necessárias.

Next.js, Tailwind, TypeScript, Framer Motion
Ignite Call

Este projeto foi desenvolvido durante o módulo 6 do bootcamp Ignite da Rocketseat. O projeto permite ao usuário criar uma conta e receber agendamentos no calendário usando a API do Google, um projeto semelhante ao Calendly.

Next.js, Next Auth, React Query, Prisma, TypeScript
